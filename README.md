# Power-System-Protection-Simulink-Design
# EECE-477: Power System Protection – Comprehensive Study Report
**Military Institute of Science & Technology (MIST)** **Department of EECE**

## 1. Project Overview
This repository contains the simulation files and analytical documentation for a comprehensive protection study of a 33/11 kV radial distribution feeder. The study focuses on fault analysis using symmetrical components, MATLAB/Simulink verification, and the selection of protection hardware (CTs and Circuit Breakers) based on calculated fault levels.

## 2. Project Objectives
* **Fault Analysis:** Calculation of 3-phase and SLG fault currents for an 11 kV radial feeder.
* **System Modeling:** Development of a MATLAB/Simulink model to simulate transient fault behavior.
* **Validation:** Verification of analytical hand calculations against simulation results.
* **Equipment Selection:** Specification of Vacuum Circuit Breaker (VCB) and Current Transformer (CT) ratings.
* **System Safety:** Designing a reliable scheme for rapid fault isolation.
